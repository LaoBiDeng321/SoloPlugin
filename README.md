# 本地服务插件管理器

一个功能完善的浏览器插件，用于在访问本地Web服务时自动管理其他浏览器插件的状态。

## 功能特性

### 核心功能

- **自动检测本地服务**：当浏览器访问用户配置列表中的本地服务（包括localhost及其他本机Web服务）时，自动检测并禁用除本插件外的所有其他已安装浏览器插件
- **本地文件支持**：支持 file:// 协议，当访问本地文件时也会自动禁用其他插件（可通过设置开关控制）
- **智能恢复插件**：当用户离开这些本地服务页面（包括关闭标签页、导航至非列表中地址或关闭浏览器）时，精确恢复之前禁用的所有插件至原始状态
- **无干扰运行**：确保在插件禁用/恢复过程中不影响目标本地Web服务的正常加载、渲染及功能使用

### 用户配置管理

- **友好的配置界面**：提供直观的配置界面，允许用户添加、编辑、删除需要应用插件禁用规则的本地服务域名或IP地址列表
- **灵活的匹配模式**：支持通配符匹配或正则表达式功能，以灵活匹配本地服务地址模式
- **开关控制**：提供开关控件允许用户临时启用/禁用本插件的自动管理功能

### 用户界面

- **状态指示器**：设计直观的插件状态指示器，清晰显示当前是否处于插件禁用模式
- **插件管理面板**：实现详细的插件管理面板，展示当前所有浏览器插件的状态信息
- **手动控制**：提供简洁的操作界面，允许用户手动触发插件禁用/恢复操作


## 安装方法

### 开发者模式安装

1. 下载或克隆本项目到本地
2. 打开Chrome或Edge浏览器
3. 访问 `chrome://extensions/`（Chrome）或 `edge://extensions/`（Edge）
4. 开启右上角的"开发者模式"
5. 点击"加载已解压的扩展程序"
6. 选择项目根目录
7. 插件安装完成

### 打包安装

1. 在浏览器扩展管理页面点击"打包扩展程序"
2. 选择项目根目录
3. 生成.crx或.zip文件
4. 将文件拖拽到浏览器中安装

## 使用说明

### 基本使用

1. 安装插件后，点击浏览器工具栏中的插件图标打开弹出窗口
2. 在弹出窗口中可以看到当前的插件状态和已禁用的插件数量
3. 点击"设置"按钮进入详细配置页面

### 配置监控域名

1. 在设置页面的"监控域名"部分，输入要监控的域名或IP地址
2. 点击"添加"按钮将域名添加到列表中
3. 支持通配符匹配，如 `*.localhost` 或 `192.168.*`
4. 如需使用正则表达式，开启"使用正则表达式"开关

### 配置 file:// 协议支持

1. 在设置页面的"基本设置"部分，找到"包含 file:// 协议"开关
2. 开启后，访问本地文件（file:// 协议）时也会自动禁用其他插件
3. 关闭后，仅对配置的域名/IP地址进行插件管理
4. 默认情况下此功能是开启的

### 启用/禁用自动管理

1. 在弹出窗口或设置页面中，找到"启用自动管理"开关
2. 开启后，插件将自动管理访问本地服务时的其他插件状态
3. 关闭后，插件将暂停自动管理功能

### 手动控制插件

1. 在弹出窗口中，点击"禁用其他插件"按钮可手动禁用所有其他插件
2. 点击"恢复所有插件"按钮可恢复所有被禁用的插件
3. 适用于需要临时控制插件状态的场景

### 主题切换

1. 在设置页面的"主题设置"部分，选择"亮色主题"或"暗色主题"
2. 主题设置会自动保存并在下次使用时应用

### 背景设置

1. 设置页面支持随机背景功能，包括图片和视频背景
2. 背景图片存储在 `img/backgrounds/` 文件夹中
3. 背景视频存储在 `videos/backgrounds/` 文件夹中
4. 每次打开或刷新设置页面时，会随机选择一张背景图片或视频
5. 支持类型选择功能，可以选择只显示图片背景、只显示视频背景或两者都显示
6. 支持自动轮转功能，可以设置背景自动切换的时间间隔
7. 支持拖动设置页面，按住头部区域即可移动
8. 如果没有背景资源，会显示默认纯色背景

## 技术实现

### Manifest V3

本插件使用Chrome Extension Manifest V3规范开发，确保与最新浏览器标准兼容。

### 核心API

- **chrome.management**：用于获取和管理浏览器插件状态
- **chrome.storage**：用于保存用户配置和插件状态
- **chrome.tabs**：用于监听标签页变化事件

### 性能优化

- 异步操作确保UI不阻塞
- 批量操作减少API调用次数
- 状态缓存减少重复查询

## 常见问题

### Q: 插件无法禁用某些插件？

A: 某些浏览器内置插件或系统插件可能无法被禁用，这是浏览器的限制。

### Q: 如何备份我的配置？

A: 配置数据保存在浏览器的本地存储中，可以通过浏览器同步功能进行备份。

### Q: 插件会影响浏览器性能吗？

A: 本插件经过性能优化，正常使用下对浏览器性能影响极小。

### Q: 支持哪些本地服务地址？

A: 支持所有本地Web服务地址，包括localhost、127.0.0.1、0.0.0.0、::1以及自定义的本地域名。同时支持 file:// 协议的本地文件访问。

### Q: 如何控制是否管理本地文件的插件？

A: 在设置页面的"基本设置"中，可以通过"包含 file:// 协议"开关来控制是否对本地文件进行插件管理。
